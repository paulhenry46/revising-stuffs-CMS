## Tech Stack
#### Back-end
- Laravel 10
- Laravel JetStream for authentication
- User permissions and roles package by Spatie
- PDF to image package by Spatie
#### Front-end
- Tailwind CSS with DaisyUI components
- Laravel Livewire
- Alpine.js
- Vanilla JS for the flashcard Quiz and Learn Mode
